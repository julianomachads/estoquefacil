# EstoqueFácil

Aplicação de página única (SPA) para controle de estoque, desenvolvida para a disciplina de Desenvolvimento Web.

Construída apenas com HTML5, CSS3 e JavaScript puro, sem uso de bibliotecas ou frameworks.

## Funcionalidades

- Cadastro e login de usuário (nome, e-mail e senha)
- Sessão e dados persistidos em localStorage
- Listagem de produtos com opções de editar e excluir
- Cadastro de produtos em janela popup com confirmação de exclusão
- Navbar fixa exibida apenas após o login
- Registro do autor do cadastro em cada produto

## Tecnologias

- HTML5
- CSS3
- JavaScript (sem dependências externas)

## Estrutura

- `index.html` - estrutura das telas
- `style.css` - estilos da aplicação
- `script.js` - lógica e manipulação do localStorage

## Como executar

Abra o arquivo `index.html` diretamente no navegador. Cadastre um usuário, faça login e gerencie os produtos.
